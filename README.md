Real Estate Lead Automation API
Python | Flask | JWT | SQLAlchemy

REST API built with Flask for managing and scoring real estate leads.

Features:
- Lead management (CRUD)
- Lead scoring system
- JWT authentication
- SQLite database
- REST API architecture

Tech Stack:
- Python
- Flask
- Flask-JWT-Extended
- SQLAlchemy
- Bcrypt

Endpoints:
POST /register
POST /login
GET /leads
POST /leads
